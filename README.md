# GBD 2021 Under-5 Meningitis Analysis

## Overview

This repository contains the analysis code supporting the manuscript:

> Global, Regional, and National Trends and Inequalities in Meningitis
> Burden Among Children Younger Than 5 Years, 1990–2021, With Projections
> to 2045: A Population-Based Modelling Study Using GBD 2021.

Manuscript ID: 9049168.

## Corresponding research output

The study evaluates temporal trends, geographic inequalities, pathogen-
specific patterns, functional impairments, and future projections of the
meningitis burden among children younger than 5 years.

## Repository contents

The R scripts cover:

- Joinpoint and temporal trend analyses
- Age-specific and pathogen-specific analyses
- Health-inequality analyses
- Frontier analyses
- Bayesian age-period-cohort projections
- Internal validation and calibration
- Generation of manuscript figures and supplementary figures

## Software requirements

The analyses were conducted using R versions 4.3.3 and 4.4.1, as indicated
in the corresponding script filenames.

Users should install the required R packages referenced at the beginning
of each script before running the analyses.

## Data source

The analyses use estimates from the Global Burden of Disease Study 2021.
Access to the underlying data is subject to the applicable data-access
procedures and terms of the Institute for Health Metrics and Evaluation.
